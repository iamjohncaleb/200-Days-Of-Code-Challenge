# NFT Marketplace

![Solidity](https://img.shields.io/badge/Solidity-0.8.20-blue)
![Ethers.js](https://img.shields.io/badge/Ethers.js-5.7.2-purple)

## Installation
```sh
npm install
```

## Testing
```sh
npx hardhat test
```
